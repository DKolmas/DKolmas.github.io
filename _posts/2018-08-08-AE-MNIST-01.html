Abstract¶

Primarily I am focused in that note to construct Autoencoder using Tensoflow. I want to see how well images can be reconstructed and how many samples is needed (I have used 15000 training samples). The structure of autoencdoer is verys simple: there are 3 hidden layers in total. The innermost layer has 128 neurons. Input and output is of 784 size. The reconstruction is done well and the example is show below. The 4x4 block of digits locatd on the left is a set of oryginal digits. The 4x4 block of digits on the rigts is a set of reconstructed digits.

def encoder(x, NetSize):
    actFunc = tf.nn.sigmoid
    #he_init = tf.contrib.layers.variance_scaling_initializer() # HE initialization for ReLu or ELU
    # Dense layer by default uses Xavier initialization
    hidden_1 = tf.layers.dense(x,NetSize['size_hid_1'], activation=actFunc, name='Enc_Hid_1')
    hidden_2 = tf.layers.dense(hidden_1,NetSize['size_hid_2'], activation=actFunc, name='Enc_Hid_2')
    hidden_3 = tf.layers.dense(hidden_2,NetSize['size_hid_3'], activation=actFunc, name='Enc_Hid_3')
    hidden_4 = tf.layers.dense(hidden_3,NetSize['size_hid_4'], activation=actFunc, name='Enc_Hid_4')
    return hidden_4
In [271]:
def decoder(x,NetSize):
    actFunc = tf.nn.sigmoid
    #he_init = tf.contrib.layers.variance_scaling_initializer() # HE initialization for ReLu or ELU
    # Dense layer by default uses Xavier initialization
    hidden_1 = tf.layers.dense(x, NetSize['size_hid_3'], activation=actFunc, name='Dec_Hid_1')
    hidden_2 = tf.layers.dense(hidden_1, NetSize['size_hid_2'], activation=actFunc, name='Dec_Hid_2')
    hidden_3 = tf.layers.dense(hidden_2, NetSize['size_hid_1'], activation=actFunc, name='Dec_Hid_3')
    hidden_4 = tf.layers.dense(hidden_3, NetSize['size_output'], activation=actFunc, name='Dec_Hid_4')
    return hidden_4

The example of test samples shows that number of iteration is not enough to correctly reconstruct all digits. For example number 8 is reconstructed to 0 and number 0 is reoconstructed to 8. This sugests that the latent space for these two number overlaps. But the overlapping is partial as we can see that other number 8 is correctly reconstructed into number 8. Another reconstruction problems appears with number 4 which is reconstructed two times into number 9.
